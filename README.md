# Sameena Nikhat — Professional Profile

Multi-page personal profile site for Sameena Nikhat, Ph.D., Postdoctoral Research Fellow at Mayo Clinic, Arizona.

## Pages

- **index.html** — Hero introduction, research focus, recent highlights
- **about.html** — Research statement, education timeline, technical skills
- **research.html** — Detailed walkthrough of postdoctoral and predoctoral research projects
- **publications.html** — Peer-reviewed publications, manuscripts in preparation, patents, conference presentations
- **achievements.html** — Awards, fellowships, and service contributions

## Stack

Plain HTML, CSS, and vanilla JavaScript — no frameworks, no build step. Designed for fast, reliable rendering via GitHub Pages.

## Local preview

```bash
cd site/
python3 -m http.server 8000
# open http://localhost:8000
```

## Deploy

This site is deployed to GitHub Pages. Pushing to the `main` branch of the user-pages repository (`<username>.github.io`) publishes automatically.
